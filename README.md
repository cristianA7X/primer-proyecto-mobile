# Primer Proyecto Mobile

## Descripción

Este proyecto corresponde a un trabajo práctico de la materia **Desarrollo de Aplicativos Móviles**. Su objetivo es poner en práctica el uso de React Native junto con Expo Go.

Se enfoca principalmente en la visualización de cambios en tiempo real, la aplicación de estilos y la maquetación de una interfaz mobile básica.

## Objetivo

Aplicar conceptos iniciales de desarrollo mobile utilizando React Native, incluyendo:
- Uso de componentes
- Aplicación de estilos
- Organización de la interfaz

## Funcionalidad

El proyecto no presenta funcionalidades dinámicas.  
Consiste en la visualización de una misma frase repetida en cuatro contenedores, cada uno con un estilo diferente, con el fin de practicar el diseño y la presentación visual.

## Tecnologías utilizadas

- React Native
- Expo (Expo Go)
